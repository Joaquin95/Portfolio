Texas Mortgage Leads

TexasMortgageLead.com is a subscription-based SaaS platform built for loan officers to acquire high-intent mortgage leads across Texas. The project leverages a modern full-stack architecture with React for the frontend UI, Firebase for real-time data management, and Stripe and PayPal for seamless subscription billing. The interface is designed to be intuitive and mobile-responsive, with accessibility and conversion-friendly design patterns that cater to professionals across devices. On the backend, the system uses Firebase Functions (serverless) to handle lead routing, subscription logic, email notifications, and monthly usage limits. Loan officers are dynamically matched with leads based on their current plan and lead consumption, ensuring fair distribution and quota enforcement. Real-time Firestore listeners power the dashboard with live lead updates, while SendGrid handles transactional alerts and quota warnings to keep users informed. The platform also features PayPal Smart Buttons embedded within the plan selector, giving users the option to upgrade or activate subscriptions using an embedded checkout flow. API keys and credential management are securely handled via environment variables and restricted access patterns to ensure proper authorization across services. This infrastructure ensures scalability, security, and a customer-first approach tailored to mortgage professionals. Whether it’s plan upgrades, lead assignment, or quota tracking, every interaction on the platform is backed by a thoughtful mix of frontend logic and cloud-native backend functions. TexasMortgageLead.com is not just a lead distribution tool—it’s a robust, real-time business platform built for reliability and ease-of-use.

Valley lending Website

The Valley Lending portfolio is a responsive landing page built with React.js to serve as a professional online presence for my current career as a mortgage loan officer. The project uses custom CSS for styling, with styles organized to maintain clean and modular code. Navigation is handled through anchor links for smooth scrolling between sections such as Home, About, Services, Apply, and Contact. An external link is integrated to allow users to start a loan application, which opens in a new tab for a seamless user experience. The site was deployed using Vercel, making it easily accessible to clients online.

Cryptogreek

I developed a cryptocurrency tracking and management web application using React.js for the frontend, with a Node.js backend, PostgreSQL for database management, and a third-party API from CoinGecko. The application allows users to view real-time cryptocurrency data such as price, market cap, and 24-hour changes for various coins, and also features user registration and login functionality with JWT authentication. I used CoinGecko API to fetch market data using a simple GET request, which provided real-time cryptocurrency information. Deployed using Render for easier deployment. The PostgreSQL database was initially hosted locally and then migrated to Render's PostgreSQL service.To manage sensitive credentials such as database URLs, API keys, and SMTP credentials, I utilized a .env file for both the frontend and backend. To ensure security, I added the .env file to .gitignore. The main challenges involved ensuring smooth deployment and troubleshooting issues related to Vite permission errors, file case sensitivity, and database migrations.(Still in working process)
