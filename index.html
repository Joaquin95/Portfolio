Best Tree Service

The Best Tree Service website is a modern, responsive web application built using React.js, enables fast and scalable user interfaces. Navigation between pages such as the homepage and the estimate form is handled seamlessly with React Router DOM, allowing for a smooth user experience without full page reloads. The project features an image carousel implemented using React Slick, which showcases examples of past work in a visually appealing slider format. Form submissions are managed through Formspree, a secure and lightweight solution that collects customer inquiries and delivers them directly to the business. Google Analytics 4 (GA4) was integrated to track visitor behavior, traffic sources, and user engagement across the site, providing valuable insights to improve marketing and performance. Designed with both user experience and business objectives in mind, the website is optimized for deployment on platforms like Vercel or Netlify, with support for HTTPS and continuous deployment pipelines.(Will be deployed on 5.26.2025).

Valley lending Website

The Valley Lending portfolio is a responsive landing page built with React.js to serve as a professional online presence for my current career as a mortgage loan officer. The project uses custom CSS for styling, with styles organized to maintain clean and modular code. Navigation is handled through anchor links for smooth scrolling between sections such as Home, About, Services, Apply, and Contact. An external link is integrated to allow users to start a loan application, which opens in a new tab for a seamless user experience. The site was deployed using Vercel, making it easily accessible to clients online.

Cryptogreek

I developed a cryptocurrency tracking and management web application using React.js for the frontend, with a Node.js backend, PostgreSQL for database management, and a third-party API from CoinGecko. The application allows users to view real-time cryptocurrency data such as price, market cap, and 24-hour changes for various coins, and also features user registration and login functionality with JWT authentication. I used CoinGecko API to fetch market data using a simple GET request, which provided real-time cryptocurrency information. Deployed using Render for easier deployment. The PostgreSQL database was initially hosted locally and then migrated to Render's PostgreSQL service.To manage sensitive credentials such as database URLs, API keys, and SMTP credentials, I utilized a .env file for both the frontend and backend. To ensure security, I added the .env file to .gitignore. The main challenges involved ensuring smooth deployment and troubleshooting issues related to Vite permission errors, file case sensitivity, and database migrations.(Still in working process)

Anime Finder

This is a group project. Anime is a web application that allows users to search for their favorite anime series and movies. The frontend is built using React, providing a responsive and user-friendly interface. The backend is developed with Node.js and Express, handling API requests and data management. The project utilizes the Jikan API to fetch anime data. I built the calendar feature, which displays users favorite anime on the day each episode airs, providing a visual schedule of upcoming shows. The app includes sign-up and login functionality, allowing users to save favorites and personalize their experience. Technologies used include React, React Bootstrap, React Router DOM, and FullCalendar for the calendar view.
